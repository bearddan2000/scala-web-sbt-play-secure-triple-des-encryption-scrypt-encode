# scala-web-sbt-play-secure-triple-des-encryption-scrypt-encode

## Description
A call to playframework web
with simple login forms scrypt hash
with 3DES encryption.

## Tech stack
- scala
- sbt
  - play
  - 3DES
  - scrypt

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`
http://localhost
- username: user
- password: pass

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
